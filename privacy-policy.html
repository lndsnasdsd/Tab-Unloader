1. Общие положения

Настоящая Политика конфиденциальности описывает, как браузерное расширение Tab Unloader ("Tab Unloader", "расширение") обрабатывает данные пользователей.

Tab Unloader предназначено только для одной цели: управления неактивными вкладками браузера для снижения потребления памяти. Расширение работает локально в браузере пользователя и не использует удаленные сервисы, учетные записи или облачную синхронизацию.

2. Какие данные обрабатывает расширение

  • Сведения об открытых вкладках, включая URL вкладок, заголовки вкладок и технические статусы вкладок, например активна ли вкладка, закреплена ли она, воспроизводит ли звук и выгружена ли она уже.
  • Пользовательские настройки, включая состояние включения расширения, время неактивности, белый список исключенных доменов и счетчик выгруженных вкладок.

3. Как используются данные

  • Для определения неактивных вкладок.
  • Для исключения активных, закрепленных, воспроизводящих звук и добавленных в белый список вкладок.
  • Для отображения списка вкладок в popup-окне.
  • Для ручной выгрузки допустимых вкладок по запросу пользователя.
  • Для сохранения пользовательских настроек между сеансами браузера.

4. Какие данные расширение не собирает

  • Персональные данные, позволяющие идентифицировать личность.
  • Финансовые и платежные данные.
  • Данные для аутентификации, включая пароли и PIN-коды.
  • Личную переписку, включая электронные письма, сообщения чатов и SMS.
  • Медицинскую информацию.
  • Точные данные о местоположении.
  • Содержимое веб-страниц, включая текст страниц, данные форм, изображения, аудио и видео.
  • Телеметрию действий пользователя, такую как клики, нажатия клавиш, движение мыши или прокрутка.

5. Локальная обработка и хранение

Расширение обрабатывает связанные с вкладками данные локально на устройстве пользователя.

  • URL вкладок, заголовки и технические статусы используются в памяти во время работы расширения.
  • Настройки и служебные данные хранятся локально в браузере через Chrome extension storage API.

Расширение не передает пользовательские данные разработчику, третьим лицам или на внешние серверы.

6. Передача и продажа данных

  • Tab Unloader не продает пользовательские данные.
  • Tab Unloader не передает пользовательские данные третьим лицам.
  • Tab Unloader не использует пользовательские данные для рекламы, профилирования или маркетинга.
  • Tab Unloader не использует удаленный код для сбора или обработки данных вне пакета расширения.

7. Срок хранения и удаление

Настройки и служебные данные хранятся локально в браузере до тех пор, пока пользователь не изменит их или не удалит расширение, с учетом поведения браузера и локального управления данными браузера.

8. Соответствие требованиям Chrome Web Store

Tab Unloader обрабатывает данные, связанные с историей веб-поиска и вкладками, только в объеме, необходимом для предоставления основной и явно описанной функции: управления вкладками и автоматической выгрузки неактивных вкладок.

Использование данных ограничено предоставлением и улучшением этой основной функции и направлено на соблюдение Политики пользовательских данных Chrome Web Store, включая требования Limited Use.

English Version

1. Overview

This Privacy Policy describes how the Tab Unloader browser extension ("Tab Unloader", "the extension") handles user data.

Tab Unloader is designed for one purpose only: managing inactive browser tabs to reduce memory usage. The extension works locally in the user's browser and does not provide any remote service, account system, or cloud synchronization.

2. Data the extension handles

  • Information about open tabs, including tab URLs, tab titles, and technical tab states such as whether a tab is active, pinned, audible, or already discarded.
  • User-defined settings, including whether the extension is enabled, the inactivity timeout, the whitelist of excluded domains, and the discarded tab counter.

3. How this data is used

  • To determine which tabs are inactive.
  • To avoid discarding active, pinned, audible, or excluded tabs.
  • To display the current tab list in the popup.
  • To allow the user to manually discard eligible tabs.
  • To save user preferences between browser sessions.

4. What the extension does not collect

  • Personally identifiable information.
  • Financial or payment information.
  • Authentication information such as passwords, credentials, or PINs.
  • Personal communications such as emails, chat messages, or SMS.
  • Health information.
  • Precise location data.
  • The contents of web pages such as page text, form entries, images, audio, or video.
  • User interaction telemetry such as clicks, keystrokes, mouse movement, or scrolling.

5. Local processing and storage

The extension processes tab-related data locally on the device.

  • Tab URLs, titles, and technical states are used in memory while the extension is running.
  • User settings and service data are stored locally in the browser using the Chrome extension storage API.

The extension does not transmit user data to the developer, to third parties, or to external servers.

6. Data sharing and sale

  • Tab Unloader does not sell user data.
  • Tab Unloader does not share user data with third parties.
  • Tab Unloader does not use user data for advertising, profiling, or marketing.
  • Tab Unloader does not use remote code to collect or process data outside the packaged extension.

7. Retention and deletion

Settings and service data remain stored locally in the browser until the user changes those settings or removes the extension, subject to browser behavior and local browser data management.

8. Chrome Web Store compliance

Tab Unloader handles tab-related browsing data only to provide its clearly described core functionality: tab management and automatic unloading of inactive tabs.

The extension's use of data is limited to providing and improving this core feature and is intended to comply with the Chrome Web Store User Data Policy, including the Limited Use requirements.
